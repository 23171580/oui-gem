# OUI

look up OUI information /w Chinese translation using MAC addresses

## Installation

Add this line to your application's Gemfile:

    gem 'oui'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install oui

## Usage

### From the command line

    $ oui 14:10:9f:ea:22:33
